Autoruns 日本語解説 - スキャン オプション

Sysinternals 日本語解説 > Autoruns > スキャン オプション

Scan Options ダイアログを使用して、どの情報をスキャンで収集するのか指定します。

Autoruns は Verify code signatures で証明書失効リスト (CRL) Web サイトを照会してイメージの署名が有効かどうか判断するようになります。Autoruns は署名がないか、システムによって信頼されるルート証明機関のリストにある証明書ルート権限によって署名されていないイメージの会社名の横に「(Not verified)」テキストを表示します。Verify code signatures オプションを選択すると、署名のないイメージに対応するエントリがピンク色で強調表示されます。Verify code signatures オプションが無効だと、会社名やファイルの説明がないイメージの項目がピンク色で強調表示されます。

Check VirusTotal.com オプションを有効にすると、Autoruns は無料の VirusTotal.com サービスに問い合わせて、エントリを数十のマルウェア対策エンジンでスキャンした結果を取得します。表示される結果は、「エントリがマルウェアだと報告したエンジンの数/スキャンしたエンジンの総数」か、エントリがスキャンのために送信されなかったことを示す「Unknown」のいずれかです。Submit Unknown Images を有効にすると Autoruns はスキャンのためにイメージを送信して結果を待ちます。これには数分間かかることがあります。

Scan only per-user locations オプションを選択すると、Autoruns は現在または指定されたユーザープロファイルのユーザーごとの場所のみをスキャンします。これは管理者ではないアカウントに影響するエントリだけを分析するのに役立ちます。
