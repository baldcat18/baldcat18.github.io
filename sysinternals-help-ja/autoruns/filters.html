Autoruns 日本語解説 - フィルター

Sysinternals 日本語解説 > Autoruns > フィルター

Options メニューの Hide Microsoft Entries、Hide Windows Entries、および Hide VirusTotal Clean Entries の使用は、インストール後にシステムに追加されたソフトウェアを確認するのに役立ちます。システムによって信頼されているファイルなのかデジタル署名で検証できない場合、Autoruns はイメージの会社名の前に「(Not verified)」を追加します。

  • Hide Microsoft Entries が有効の場合、Verify Image が選択されている時は Microsoft によって署名されたイメージが非表示になり、Verify Image が選択されていない時はリソースの会社名フィールドに Microsoft が含まれているイメージが非表示になります。
  • Hide Windows Entries は、Verify Image が選択されている時に Windows によって署名されたイメージを非表示にします。Verify Image が選択されていない時、Hide Windows Entries はリソースの会社名フィールドに Microsoft が含まれ %SystemRoot% ディレクトリ内にあるイメージを非表示にします。
  • Hide VirusTotal Clean Entries は VirusTotal で検出率 0 のエントリを非表示にします。

ツールバーには、表示する項目を動的にフィルタリングする自由形式のテキストエントリがあります。フィルターを取り消すには Filter エントリをクリアするだけです。
