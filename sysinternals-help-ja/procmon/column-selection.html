Process Monitor 日本語解説 - 列の選択

Sysinternals 日本語解説 > Process Monitor > 列の選択

列をドラッグして順番を並べ替えることができます。表示する列をカスタマイズするには、[Options] メニューから [Select Columns] を選択して [Process Monitor Column Selection] ダイアログを開きます。選択可能な列には次のものが含まれます。

Application Details

  • Process Name イベントを発生させたプロセスの名前。
  • Image Path プロセスで実行されているイメージのフルパス。
  • Command Line プロセスの起動に使われたコマンドライン。
  • Company Name プロセスのイメージファイルに埋め込まれた [会社] のテキスト。このテキストは、アプリケーション開発者によって任意で定義されます。
  • Description プロセスのイメージファイルに埋め込まれた [ファイルの説明] のテキスト。このテキストは、アプリケーション開発者によって任意で定義されます。
  • Version プロセスイメージファイルに埋め込まれた [製品バージョン] 番号。この情報は、アプリケーション開発者によって任意で指定されます。

Event Details

  • Sequence Number 現在のフィルターに含まれるすべてのイベントに関する操作の相対位置。
  • Event Class イベントクラス (File System、Registry、Process など)。
  • Operation イベントの具体的な動作 (たとえば Read、RegQueryValue など)。
  • Date & Time 操作の日付と時刻の両方。
  • Time of Day 操作の時刻のみ。
  • Path イベントが参照するリソースのパス。
  • Detail イベントに固有の追加情報。
  • Result 完了した操作のステータスコード。
  • Relative Time Process Monitor の開始または Process Monitor の表示が最後にクリアされた時刻に対する操作の相対時間。
  • Duration 操作が完了するまでにかかった時間。

Process Management

  • User Name 操作を実行したプロセスの実行ユーザーの名前。
  • Session ID 操作を実行したプロセスが実行されている Windows セッション。
  • Authentication ID 操作を実行したプロセスが実行されているログオンセッション。
  • Process ID 操作を実行したプロセスのプロセス ID (PID)。
  • Thread ID 操作を実行したスレッドのスレッド ID (TID)。
  • Integrity Level 操作を実行したプロセスの整合性レベル。
  • Virtualized 操作を実行したプロセスの UAC 仮想化状態。
