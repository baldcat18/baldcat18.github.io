Process Monitor 日本語解説 - 紹介

Sysinternals 日本語解説 > Process Monitor > 紹介

Process Monitor はファイルシステム、レジストリ、プロセス/スレッドのアクティビティをリアルタイムで表示する Windows 用の高度な監視ツールです。Filemon と Regmon というレガシーな 2 つの Sysinternals ユーティリティの機能を兼ね備え、高度で非破壊的なフィルタリング、セッション ID やユーザー名などの包括的なイベントプロパティ、信頼性の高いプロセス情報、各操作ごとの統合シンボルサポートを備えた完全なスレッドスタック、ファイルへの同時ロギングなどを詳細に記録します。Process Monitor はその唯一で強力な機能で、システムのトラブルシューティングやマルウェア解析ツールキットの中核となるユーティリティになっています。
