Process Explorer 日本語解説 - Users メニュー

Sysinternals 日本語解説 > Process Explorer > Users メニュー

Process Explorer は現在接続しているセッションをリストアップする Users メニューを表示します。Process Explorer はセッションのセッション ID とそのセッションにログインしているユーザーを名前とするメニュー項目をセッションごとに作成します。各項目は、切断、ログオフ、およびそのセッションのユーザーへのメッセージ送信のためのオプションを持つサブメニューを開きます。さらに各セッション項目の Properties メニューでは、セッションに接続しているクライアントの IP アドレスや名前など、セッションに関する詳細情報を表示するダイアログボックスが開きます。

Users メニューの内容は、現在のセッション情報を反映させるためにメニューを開くたびに更新されます。
