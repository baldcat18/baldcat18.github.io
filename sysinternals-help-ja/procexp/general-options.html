Process Explorer 日本語解説 - 色々なオプション

Sysinternals 日本語解説 > Process Explorer > 色々なオプション

コマンドライン オプション

/e管理者権限無しで起動したときに、管理者として実行しなおすための UAC 昇格プロンプトを表示します。
/s:<pid>起動後に指定されたプロセス ID を持つプロセスを選択状態にします。
/tProcess Explorer を最小化状態で起動し、通知領域に表示します。
/p:[r|h|n|l]Process Explorer の基本優先度をリアルタイム (r)、高 (h)、通常 (n)、低 (l) に設定します。

Always on Top: この項目を選ぶと、Process Explorer のウィンドウが他のウィンドウより手前に表示されるようになります。

Replace Task Manager: [Options] メニューのこの項目を選択すると、タスク マネージャーを起動するときに Process Explorer が代わりに実行されます。これはタスク マネージャーの起動方法に関係なく、すべてのユーザーに影響するグローバル設定です。タスク マネージャーを置き換えるとメニュー項目の名前が [Restore Task Manager] に変更され、それを選択すると Process Explorer の関連付けが解除されます。

Hide When Minimized: [Options] メニューにあるこの項目を選択すると、Process Explorer を最小化したときに現在の CPU 使用率を反映する小さなグラフを通知領域に表示します。CPU使用率が 70% 未満の場合、メーターは緑色で表示されます。70% から 90% の間だと黄色で表示されます。90% を超えると赤で表示されます。CPU 使用率グラフは、現在定義されている更新間隔で更新されます。Process Explorer を通知領域から開始するには、コマンドライン引数として /t オプションを指定します。Process Explorer の通知領域アイコンをシングルクリックすると、通知領域内に最小化されているかどうかに関係なく、ウィンドウを復元してフォアグラウンドに表示します。

Allow Only One Instance: これを選択すると、Process Explorer のインスタンスが複数同時に実行されないようになります。

Confirm Kill: この設定を無効化すると、Process Explorer でプロセスを強制終了する時に確認のダイアログを表示しないようになります。

Tray Icons: このメニューで、CPU、I/O、ディスク、ネットワーク、GPU、コミットチャージ、物理メモリの各履歴の中でどれを通知領域アイコンとして表示するか設定できます。

Verify Image Signatures: この設定が有効な時にプロセスのプロパティを表示すると、プロセスに対応するイメージファイルのデジタル署名の信頼性が自動的にチェックされ、結果をプロセスのプロパティ ダイアログの company フィールドの隣に表示します。「(Verified)」はファイルが信頼できるルート認証局によって署名されていることを意味し、「(Unable to Verify)」はファイルが署名されていないか、署名の確認に失敗したことを表しています。プロセスイメージのプロパティを表示するときのパフォーマンスを向上させるには、このオプションのチェックを外します。

Configure Symbols: Process Explorer でプロセスのプロパティ ダイアログのスレッド タブとスレッドスタック ウィンドウのスレッド開始アドレスを解決するには、まず Microsoft の Web サイトから Debugging Tools for Windows パッケージをダウンロードし、既定のディレクトリにインストールしてシンボルを構成します。Configure Symbols ダイアログを開いて、デバッグツール ディレクトリにある dbghelp.dll のパスを指定し、シンボルパスを入力して、シンボル エンジンが要求に応じて Microsoft のサーバーからシンボルをディスク上のディレクトリにダウンロードできるようにします。たとえば、シンボルを c:\symbols ディレクトリにダウンロードするには、この文字列を入力します。

srv*c:\symbols*http://msdl.microsoft.com/download/symbols

Difference Highlight Duration: このダイアログで、新しいプロセスを緑色で表示し、終了したプロセスを赤色で表示する時間を設定できます。デフォルトは 1 秒です。強調表示の色は [Options] メニューにある [Configure Colors] ダイアログで変更できます。
