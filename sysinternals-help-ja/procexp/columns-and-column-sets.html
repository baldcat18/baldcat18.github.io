Process Explorer 日本語解説 - 列と列セット

Sysinternals 日本語解説 > Process Explorer > 列と列セット

列の選択

Process Explorer のメインウィンドウに表示される情報は十分にカスタマイズ可能です。列を新しい位置にドラッグして並べ替えることができます。各ビューおよびステータス バーに表示するデータ列を選択するには、View|Select Columns を選択するか、列ヘッダーを右クリックしてコンテキストメニューから Select Columns を使用します。Select Columns エディターが開くので、プロセス、DLL、ハンドル ペイン、およびステータス バーに表示する列を選択します。

列セット

View|Save Column Set を選択すると、列の構成および関連する並び順の設定を保存できます。Process Explorer が列の名前を設定するダイアログを表示します。保存された列セットを表示するには、View|Load Column Set メニューを選択するか、関連するアクセラレーター キーを入力します。View|Organize Column Sets を選択すると Column Set Organizer ダイアログが開き、既存の列セットを並べ替えたり名前を変更したりできます。
