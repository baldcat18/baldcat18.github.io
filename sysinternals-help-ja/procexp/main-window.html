Process Explorer 日本語解説 - メイン ウィンドウ

Sysinternals 日本語解説 > Process Explorer > メイン ウィンドウ

ビュー

Process Explorer ウィンドウは、デフォルトで2つのペインを表示します。上部ペインは常にプロセス リストです。下部には上部ペインで選択したプロセスがロードした DLL のリスト、またはプロセスが開いているオペレーティング システムのリソース ハンドル (ファイル、レジストリ キー 、同期オブジェクト) のどちらかを表示します。ビューモードは、どの情報を下部ペインに表示するかを決定します。ビューを切り替えるには、View|Lower Pane View メニュー項目、対応するツールバーボタン (トグル)、または Ctrl + D (DLL ビュー)、Ctrl-H (ハンドルビュー) アクセラレーター キーを使用します。

システム上で実行中のプロセスのみを表示したい場合は、View|Hide Lower Pane や対応するツールバー ボタンを選択、Ctrl + L アクセラレーター、ペインの境界部分を Process Explorer ウィンドウの最下部までドラッグすることで、下部ペインを隠すことができます。View|Show Lower Pane を選択、Ctrl + L を押す、またはツールバー ボタンをもう一度選択することで、下部ペインを元に戻すことができます。

ミニグラフ

Process Explorer のメインウィンドウの最上部にはツールバーと、CPU、メモリ、および I/O 履歴を表示するミニグラフが含まれています。それらは互いにリサイズしたり、ドラッグして別の行に移動できます。ミニグラフにはシステムの動作履歴が表示され、グラフの上にマウスポインターを置くと、カーソルがある時点でのプロセス情報を関連する時刻と共にツールチップに表示されます。たとえば、ミニ CPU グラフのツールチップには、CPU を最も消費しているプロセスが表示されます。いずれかのミニグラフをクリックすると、System Information ダイアログが開きます。

更新頻度と差分強調表示

Process Explorer のウィンドウを更新する速度を設定するには、View|Update Speed メニュー項目を使用します。View|Refresh Now、ツールバーのRefresh Now ボタン、または F5 キーを押すことでいつでも手動でビューを更新できます。プロセスがジョブ オブジェクトの一部か、.NET ランタイムを使用しているかのような、いくつかの検査はプロセス起動時にのみ行われます。F5 キーを押すことで、すべてのプロセスの状態を再確認できます。

Process Explorer は差分の強調表示を使用して、更新の間に項目がどのように変化したかを確認できます。プロセス、DLL、およびハンドルでは、終了したり閉じられた項目は赤で表示され、新しい項目は緑で表示されます。更新を一時停止していない場合、Options|Difference Highlight Duration ダイアログで指定された時間 (デフォルト値は 1 秒)だけ強調表示されます。表示を一時停止すると、次回手動で更新するまで違いが強調表示されたままになります。

不透過率

View|Opacity メニュー下の項目を選択することで、Process Explorer ウィンドウがある程度透明になり、下にあるウィンドウが表示されるようにできます。

保存

File|Save を選択すると、プロセスと下部ペイン (表示されている場合) の内容をタブ区切りのテキストファイルとして保存します。

シャットダウンとログオフ

File|Shutdown メニュー項目を使って、システムのシャットダウン、再起動、ロック、またはログオフできます。休止状態とスタンバイ (スリープ) のオプションも利用可能なら提供されます。

Run

このオプションでは、Windows 標準の [ファイル名を指定して実行] ダイアログを使って Process Explorer から他のアプリケーションを実行できます。

Run as Administrator

これは [Run] の亜種であり 、アプリケーションを管理者権限で起動できます。

Run as Limited User

これは [Run] の亜種であり 、指定したアプリケーションをProcess Explorer を実行しているのと同じアカウントで管理者権限を与えずに実行します。このオプションは、信頼されていないデータへのアクセスによって攻撃される可能性のある Internet Explorer などのアプリケーションにシステムを公開するのを制限します。
