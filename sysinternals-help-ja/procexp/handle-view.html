Handle コンテキストメニュー

[Handle] メニューの下か、右クリックした時の [Handle] のコンテキストメニューに 2 つの項目が表示されます。

Close Handle: この項目を選択すると、ハンドルを強制的に閉じます。これは自己責任で使用してください。ハンドルを所有しているプロセスはそのハンドルが閉じられたことを認識しないため、この機能を使用するとアプリケーションがクラッシュしたりデータが破損する可能性があります。システムプロセスのハンドルを閉じるとシステムがクラッシュする可能性があります。

Properties: この項目を選択すると、 Process Explorer はオブジェクトで開いているハンドルの数やオブジェクトへのカーネル参照の数を表示するハンドル プロパティダイアログを開きます。またオブジェクトの種類に固有の情報も表示します。たとえば mutant オブジェクトのプロパティを表示すると、Process Explorer は mutant が所有されているかどうか、もしそうならどのスレッドが所有しているかを報告します。

ハンドル プロパティダイアログの セキュリティ タブには、ハンドルが参照しているオブジェクトに適用されるセキュリティが表示されます。

名前のないハンドルを表示

デフォルトでは、 Process Explorer は名前を持つオブジェクトへのハンドルのみを表示します。View メニューの Show Unnamed Handles and Mappings を選択すると、Process Explorer は選択されているプロセスが開いているハンドルを名前のないオブジェクトも含めてすべて一覧表示します。このオプションを選択すると Process Explorer は CPU リソースを著しく消費することに注意してください。
