Process Explorer 日本語解説 - 概要

Sysinternals 日本語解説 > Process Explorer > 概要

Process Explorer は、タスク マネージャーが省く情報を取得できる高度なプロセス管理ユーティリティです。アイコン、コマンド ライン、イメージのフルパス、メモリ統計、ユーザー アカウント、セキュリティ属性など、プロセスに関する詳細情報を表示します。特定のプロセスにズームインすると、そのプロセスがロードした DLL や開いているオペレーティング システムのリソース ハンドルを一覧表示できます。検索機能を使用すると、ファイル、ディレクトリ、レジストリ キーなどのリソースを開いているプロセスを見つけたり、DLL をロードしているプロセスの一覧を表示できます。

Process Explorer の表示は、2つのサブ ウィンドウで構成されています。上部には、現在アクティブなプロセスのリストを、プロセスを所有するアカウントの名前も含めて常に表示しますが、閉じることができる下部ウィンドウに表示される情報は、Process Explorer のモードによって異なります。ハンドル モードでは、上部のウィンドウで選択したプロセスが開いているハンドルを表示します。Process Explorer が DLL モードの場合、プロセスがロードした DLL とメモリ マップド ファイルを表示します。

Process Explorer には、特定のハンドルを開いていたり DLL をロードしているプロセスをすばやく表示する、強力な検索機能があります。Process Explorer のユニークな機能は、DLL バージョンの問題やハンドルのリークを発見したり、Windows とアプリケーションの動作を把握するのに役立ちます。

Sysinternals の Web サイトで、同等の機能を持つコマンドライン ツールである Handle と ListDLL を入手できます。

Process Explorer は実行に管理者権限を要求しません。
