diskpart のサブコマンドと PowerShell の比較表

コマンドプロンプトと PowerShell の比較表 > diskpart

まったく同じ挙動をするとは限りません。
詳細は各機能のリファレンスで確認してください。

diskpartPowerShell説明
activeSet-Partition -IsActiveパーティションをアクティブに設定する
inactiveパーティションを非アクティブに設定する
assignAdd-PartitionAccessPathパーティションにドライブ文字やマウントポイントを割り当てる
attach vdiskMount-DiskImageVirtual Hard Disk (VHD) をマウントする
attributes diskGet-Diskディスクの属性を取得する
Set-Diskディスクの属性を設定する
attributes volumeGet-Partitionパーティションの属性を取得する
Set-Partitionパーティションの属性を設定する
cleanClear-Diskパーティションとボリュームを消去する
convertSet-Disk -PartitionStyleパーティションの形式を変換する
create partitionNew-Partitionパーティションを作成する
create volumeNew-Volumeボリュームを作成する
delete partitionRemove-Partitionパーティションを削除する
detach vdiskDismount-DiskImageVirtual Hard Disk (VHD) をアンマウントする
detail diskGet-Diskディスクを取得する
detail partitionGet-Partitionパーティションを取得する
detail volumeGet-Volumeボリュームを取得する
detail vdiskGet-DiskImageVirtual Hard Disk (VHD) を取得する
extendResize-Partitionパーティションのサイズを拡張する
shrinkパーティションのサイズを縮小する
formatFormat-Volumeボリュームをフォーマットする
offline diskSet-Disk -IsOfflineディスクをオフラインにする
online diskディスクをオンラインにする
offline volumeSet-Partition -IsOfflineパーティションをオフラインにする
online volumeパーティションをオンラインにする
removeRemove-PartitionAccessPathパーティションのドライブ文字やマウントポイントを割り当て解除する
