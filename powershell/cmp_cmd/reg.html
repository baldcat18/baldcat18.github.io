reg のオプションと PowerShell の比較表

コマンドプロンプトと PowerShell の比較表 > reg

まったく同じ挙動をするとは限りません。
詳細は各機能のリファレンスで確認してください。

regPowerShell説明
reg addNew-Itemレジストリ キーを作成する
reg add /v
reg add /ve		New-ItemPropertyレジストリ エントリを作成する
Set-ItemPropertyレジストリ エントリのデータを設定する
reg compareCompare-Objectレジストリ エントリを比較する
reg copyCopy-Itemレジストリ キーをコピーする
reg deleteRemove-Itemレジストリ キーを削除する
reg delete /v
reg delete /ve		Remove-ItemPropertyレジストリ エントリを削除する
reg queryGet-ChildItemレジストリ サブキーを取得する
reg query /v
reg query /ve		Get-ItemPropertyレジストリ エントリを取得する
