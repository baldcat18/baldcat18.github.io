netsh のサブコマンドと PowerShell の比較表

コマンドプロンプトと PowerShell の比較表 > netsh

まったく同じ挙動をするとは限りません。
詳細は各機能のリファレンスで確認してください。

netshPowerShell説明
advfirewall firewall add rule
firewall add		New-NetFirewallRule新しい受信または送信のファイアウォール規則を追加する
advfirewall firewall delete rule
firewall delete		Remove-NetFirewallRuleすべての一致するファイアウォール規則を削除する
advfirewall firewall set rule
firewall set		Set-NetFirewallRule既存の規則のプロパティに新しい値を設定する
advfirewall firewall show rule
firewall show		Get-NetFirewallRule指定されたファイアウォール規則を取得する
netsh interface ipv4 show addresses
netsh interface ipv6 show addresses		Get-NetIPAddressIP アドレスの構成を取得する
