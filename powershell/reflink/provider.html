PowerShell リファレンスへのリンク - プロバイダーのヘルプ トピック

PowerShell リファレンスへのリンク > プロバイダーのヘルプ トピック

PowerShell と共にインストールされるプロバイダーのヘルプ トピックへのリンク集です。

Alias プロバイダー
PowerShell エイリアスとそれが表す値へのアクセスを提供します。
Certificate プロバイダー
PowerShell での X.509 証明書ストアと証明書へのアクセスを提供します。
Environment プロバイダー
Windows 環境変数へのアクセスを提供します。
FileSystem プロバイダー
ファイルとディレクトリへのアクセスを提供します。
Function プロバイダー
PowerShell で定義されている関数へのアクセスを提供します。
Registry プロバイダー
PowerShell のレジストリキー、エントリ、および値へのアクセスを提供します。
Variable プロバイダー
PowerShell 変数とその値へのアクセスを提供します。
WSMan プロバイダー
Web Services for Management (WS-Management) 構成情報へのアクセスを提供します。
